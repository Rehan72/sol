### Prerequisites

- Node.js (Latest LTS recommended)
- Bun (or npm/yarn)

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the frontend directory:

   ```bash
   cd frontend
   ```

3. Install dependencies:

   ```bash
   bun install
   # or
   npm install
   ```

4. Start the development server:
   ```bash
   bun dev
   # or
   npm run dev
   ```
